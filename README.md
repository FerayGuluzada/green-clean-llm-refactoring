# LLM Refactoring

Replication package for the paper:
"Balancing Green and Clean Code: Prompting for LLM-Based Refactoring"

## Contents

- `original_scripts/` — Original Python scripts
- `refactored_scripts/` — LLM-generated refactored scripts
- `prompts/` — Prompt templates used in experiments
- `results/` — Experimental metrics and analysis results
